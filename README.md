# VinylScan Website

Static marketing site and legal pages for VinylScan.

## Pages
- `/` landing page
- `/privacy.html` privacy policy
- `/terms.html` terms of use
- `/support.html` support

## Run locally
```bash
python3 -m http.server 4020
```
Then open `http://localhost:4020`.

## Deploy
Deploy with Cloudflare Pages (Upload assets or GitHub-connected project).
